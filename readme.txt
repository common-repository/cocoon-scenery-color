Cocoonテーマで定義されているサイトのカラーを設定します。
※Cocoonテーマ専用です。Cocoonテーマ以外では使用できません。

== Description ==
Cocoonで定義しているカラーなどの情報を簡単に設定することができます。
設定したカラーなどの情報をファイルとして保存することができます。
ファイルとして保存した情報をもとにあらたなファイルを作成することもできます。

== Installation ==
1. WordPressの管理画面から [プラグイン] → [新規追加] をクリックします。
2. [プラグインの検索...]ボックスに「Cocoon Scenery Color」と入力します。
3. 「Cocoon Scenery Color」プラグインの [今すぐインストール] をクリックします。
4. プラグインページで "Cocoon Scenery Color" を有効にします。
5. [Cocoon 設定] の [サイトカラー設定] をクリックして、カラーを作成、管理することができます。
または、
1. このページからプラグインをダウンロードします。
2. WordPressの管理画面から、 [プラグイン] → [プラグインのアップロード] をクリックします。
3. [ファイルを選択] をクリックして、ダウンロードしたzipファイルをアップロードします。
4. [今すぐインストール] をクリックします。
5. プラグインページで "Cocoon Scenery Color" を有効にします。
6. [Cocoon 設定] の [サイトカラー設定] をクリックして、カラーを作成、管理することができます。
